About This Project

This website explains the basic components of computer hardware including memory, storage devices, expansion cards, and computer ports. It is designed to help students understand how different hardware parts work together in a computer system through a clear, structured, and visually engaging multi-page website.

What You Will Learn

  • Memory Systems – ROM, RAM, PROM, EPROM, EEPROM
  • Storage Devices – HDD, Disk Drives, CD, DVD
  • Expansion Components – PCI, Graphics Card
  • Computer Ports – USB, HDMI, Ethernet, VGA, Type-C
