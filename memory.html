Memory Systems

Where data lives. Where speed is born.

ROM (Read Only Memory)

Permanent memory that stores startup instructions and essential system programs.

RAM (Random Access Memory)

Temporary memory that stores data and programs currently in use by the computer.

PROM (Programmable Read Only Memory)

Memory that can be programmed once after manufacturing and then becomes permanent.

EPROM (Erasable Programmable Read Only Memory)

Memory that can be erased using ultraviolet light and reprogrammed.

EEPROM (Electrically Erasable Programmable Read Only Memory)

Memory that can be erased and reprogrammed electronically multiple times.
